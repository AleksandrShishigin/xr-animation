Анимации model-viewer

Файл GLB не меняется. Нужен только index.html. Ниже два примера из задания: autoplay и автоматический переход между клипами.

1. Autoplay

Сразу запускается анимация climb.

2. Автоматический переход между клипами

По кнопке запускается climb, затем автоматически переключается на idle с плавным переходом.

Если длительность climb у тебя другая, поменяй только число в CLIMB_DURATION.
