Анимации для вашего GLB

Исправил страницу полностью. Теперь анимации действительно запускаются через управление из JavaScript. В примере 1 по загрузке автоматически стартует climb. В примере 2 по кнопке запускается climb один раз, после чего модель плавно переходит в idle.

1. Автозапуск climb

После загрузки страницы автоматически запускается клип climb. Кнопкой можно перезапустить его вручную.

анимация: climb автозапуск: да
Ожидание загрузки модели…

2. climb → idle c crossfade

По кнопке сначала проигрывается climb один раз, затем автоматически включается idle с плавным переходом через animation-crossfade-duration.

сначала: climb потом: idle crossfade: 900 ms
Нажмите кнопку, чтобы запустить последовательность.
